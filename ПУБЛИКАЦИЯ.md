# 🚀 Публикация игры на Vercel

## Что такое Vercel?

Vercel - это бесплатный хостинг для Next.js приложений. Он автоматически:
- Развертывает ваш проект
- Обеспечивает HTTPS
- Предоставляет домен вида `your-project.vercel.app`
- Обновляет сайт при каждом изменении кода

## Шаг 1: Создание аккаунта на Vercel

1. Перейдите на [vercel.com](https://vercel.com)
2. Нажмите **"Sign Up"**
3. Войдите через GitHub, GitLab или Bitbucket
4. Подтвердите email

## Шаг 2: Подготовка проекта к деплою

### 2.1. Создайте репозиторий на GitHub

1. Перейдите на [github.com](https://github.com) и войдите
2. Нажмите **"+"** → **"New repository"**
3. Заполните:
   - **Repository name**: `epoch-of-winds`
   - **Description**: `Многопользовательская пошаговая бизнес-симуляция`
   - Выберите **Public** (для бесплатного деплоя)
4. Нажмите **"Create repository"**

### 2.2. Загрузите проект на GitHub

Откройте командную строку в папке `epoch-of-winds-new` и выполните:

```bash
# Инициализация git
git init

# Добавление всех файлов
git add .

# Первый коммит
git commit -m "Initial commit: Epoch of Winds game"

# Добавление удаленного репозитория (замените YOUR_USERNAME на ваш логин GitHub)
git remote add origin https://github.com/YOUR_USERNAME/epoch-of-winds.git

# Отправка на GitHub
git branch -M main
git push -u origin main
```

## Шаг 3: Деплой на Vercel

### Вариант A: Через GitHub (рекомендуется)

1. Войдите в [vercel.com](https://vercel.com)
2. Нажмите **"Add New..."** → **"Project"**
3. Выберите репозиторий `epoch-of-winds` из списка
4. Нажмите **"Import"**
5. Настройки проекта:
   - **Framework Preset**: Next.js (определится автоматически)
   - **Root Directory**: `./` (оставьте как есть)
   - **Build Command**: `npm run build` (автоматически)
   - **Output Directory**: `.next` (автоматически)
6. Нажмите **"Deploy"**

Подождите 1-2 минуты. После завершения вы увидите:
```
✅ Deployed!
```

### Вариант B: Через Vercel CLI

1. Установите Vercel CLI:
```bash
npm install -g vercel
```

2. Войдите в Vercel:
```bash
vercel login
```

3. Разверните проект:
```bash
vercel
```

4. Следуйте инструкциям в терминале

## Шаг 4: Настройка переменных окружения

После деплоя добавьте переменные окружения:

1. Откройте проект на Vercel
2. Перейдите в **Settings** → **Environment Variables**
3. Добавьте переменные:

| Name | Value |
|------|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | Ваш URL из Supabase |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Ваш anon ключ из Supabase |

4. Нажмите **"Save"**
5. Переразверните проект (**Deployments** → **Redeploy**)

## Шаг 5: Получение ссылки

После успешного деплоя вы получите ссылку вида:
```
https://epoch-of-winds-xyz.vercel.app
```

Эту ссылку можно отправлять другим игрокам!

## 🔄 Обновление игры

Для обновления игры:

1. Внесите изменения в код
2. Сохраните файлы
3. В командной строке:
```bash
git add .
git commit -m "Описание изменений"
git push
```

Vercel автоматически развернет новую версию!

## 📊 Мониторинг

На Vercel вы можете:
- Просматривать логи (Logs)
- Анализировать производительность (Analytics)
- Настраивать домен (Domains)
- Управлять переменными окружения (Environment Variables)

## 🔧 Решение проблем

### Ошибка "Build failed"
- Проверьте, что все зависимости установлены (`npm install`)
- Убедитесь, что `.env.local` не добавлен в git (он в `.gitignore`)

### Ошибка "Environment variables not found"
- Добавьте переменные в Settings → Environment Variables
- Переразверните проект

### Сайт не работает
- Проверьте логи в Vercel (Deployments → Ваш деплой → View Logs)
- Убедитесь, что Supabase настроен правильно

## 🌐 Собственный домен

Если хотите использовать свой домен:

1. Купите домен (например, на reg.ru, nic.ru)
2. В Vercel: Settings → Domains → Add Domain
3. Следуйте инструкциям для настройки DNS
